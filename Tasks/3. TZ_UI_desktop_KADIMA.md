## 1. Входные данные системы (что UI должен отображать)

### 1.1 Конфигурация (из `pipeline/config.py`)

UI читает/пишет PipelineConfig:

```python
PipelineConfig:
  profile: Profile            # PRECISE | BALANCED | RECALL
  modules: List[str]          # 22 модуля (NLP_MODULES | GENERATIVE_MODULES)
  thresholds: ThresholdsConfig:
    min_freq: int              # default 2 (balanced), 1 (recall), 3 (precise)
    pmi_threshold: float       # default 3.0, 5.0, 1.5
    hapax_filter: bool         # default true, false, true
  annotation: AnnotationConfig:
    label_studio_url: str      # "http://localhost:8080"
    label_studio_api_key: Optional[str]
    ml_backend_url: str        # "http://localhost:9090"
  llm: LLMConfig:
    enabled: bool
    server_url: str            # "http://localhost:8081"
    model: str                 # "dictalm-3.0-1.7b-instruct"
    max_tokens: int            # 1–8192
    temperature: float         # 0.0–2.0
  kb: KBConfig:
    enabled: bool
    embedding_model: str       # "neodictabert"
    auto_generate_definitions: bool
  logging: LoggingConfig
  storage: StorageConfig
```

13 generative sub-configs: DiacritizerConfig, TranslatorConfig, TTSConfig, STTConfig, NERConfig, SentimentConfig, SummarizerConfig, QAConfig, MorphGenConfig, TransliteratorConfig, GrammarConfig, KeyphraseConfig, ParaphraseConfig.

### 1.2 Результаты pipeline (из `engine/base.py`)

```python
ProcessorResult:
  module_name: str             # "ngram_extractor"
  status: ProcessorStatus      # READY | RUNNING | FAILED | SKIPPED
  data: Any                    # NgramResult | NPChunkResult | и т.д.
  metadata: Dict[str, Any]
  errors: List[str]
  processing_time_ms: float

PipelineResult:
  corpus_id: int
  profile: str
  module_results: Dict[str, ProcessorResult]
  terms: List[Any]             # Term objects
  ngrams: List[Any]            # Ngram objects
  np_chunks: List[Any]         # NPChunk objects
  total_time_ms: float
  status: ProcessorStatus
```

### 1.3 API schemas (из `api/schemas.py`)

UI использует те же Pydantic модели:

| Модель | Поля | UI View |
|---|---|---|
| `CorpusResponse` | id, name, language, created_at, status, document_count, token_count | Dashboard, Corpora |
| `TermResponse` | surface, canonical, kind, freq, doc_freq, pmi, llr, dice, rank | Results — Terms tab |
| `PipelineRunResponse` | run_id, corpus_id, profile, status, terms[], ngram_count, total_time_ms | Pipeline, Results |
| `ValidationReportResponse` | corpus_id, status, checks[], summary{} | Validation |
| `KBTermResponse` | id, surface, canonical, lemma, pos, definition, freq, related_count | KB |
| `KBRelationResponse` | related_term_id, related_surface, relation_type, similarity_score | KB — detail |
| `AnnotationProjectResponse` | id, name, type, ls_project_id, task_count, completed_count | Annotation |
| `LLMStatusResponse` | loaded, server_url, model | LLM |

**Threading:**
```python
class PipelineWorker(QRunnable):
    """Pipeline runs in QThreadPool — НЕ в main thread."""
    def run(self):
        result = orchestrator.run_on_text(text, config)
        self.signals.finished.emit(result)  # → main thread обновляет UI
```

**Сигналы:**
- `run_clicked` → PipelineWorker → QThreadPool
- `progress_updated(int, str)` → обновить progress bar
- `run_finished(PipelineResult)` → switch to Results view
- `run_failed(str)` → ErrorDialog

**Горячие клавиши:** F5 = Run, Esc = Stop

```python
class ResultColorDelegate(QStyledItemDelegate):
    def initStyleOption(self, option, index):
        super().initStyleOption(option, index)
        result = index.data(Qt.ItemDataRole.DisplayRole)
        if result == "PASS": option.backgroundBrush = QColor("#22c55e")
        elif result == "WARN": option.backgroundBrush = QColor("#eab308")
        elif result == "FAIL": option.backgroundBrush = QColor("#ef4444")
```

### 5.3 Threading rules

```python
# Pipeline — всегда в QThreadPool, НЕ в main thread
from PyQt6.QtCore import QRunnable, QThreadPool

class PipelineWorker(QRunnable):
    def __init__(self, orchestrator, text, config):
        self.orchestrator = orchestrator
        self.text = text
        self.config = config
        self.signals = WorkerSignals()
    
    def run(self):
        try:
            result = self.orchestrator.run_on_text(self.text, self.config)
            self.signals.finished.emit(result)
        except Exception as e:
            self.signals.error.emit(str(e))

# Generative — тоже в QThreadPool
class GenerativeWorker(QRunnable):
    def __init__(self, module, input_data):
        ...
    def run(self):
        result = self.module.process(input_data, {})
        self.signals.finished.emit(result)
```

### 10.12 PipelineWorker — threading спецификация

```python
from PyQt6.QtCore import QRunnable, QObject, pyqtSignal

class WorkerSignals(QObject):
    finished = pyqtSignal(object)   # PipelineResult
    progress = pyqtSignal(int, str) # (step, module_name)
    error = pyqtSignal(str)

class PipelineWorker(QRunnable):
    def __init__(self, orchestrator: PipelineService, text: str = None, corpus_id: int = None):
        super().__init__()
        self.orchestrator = orchestrator
        self.text = text
        self.corpus_id = corpus_id
        self.signals = WorkerSignals()
        self.setAutoDelete(True)
    
    def run(self):
        try:
            if self.text:
                result = self.orchestrator.run_on_text(self.text)
            else:
                result = self.orchestrator.run(self.corpus_id)
            self.signals.finished.emit(result)
        except Exception as e:
            self.signals.error.emit(str(e))

# В Pipeline View:
self.thread_pool = QThreadPool()
worker = PipelineWorker(self.orchestrator, text=input_text)
worker.signals.finished.connect(self._on_pipeline_finished)
worker.signals.error.connect(self._on_pipeline_error)
self.thread_pool.start(worker)
```

```python
class TableContextMenu:
    def __init__(self, table: QTableView):
        table.setContextMenuPolicy(Qt.ContextMenuPolicy.CustomContextMenu)
        table.customContextMenuRequested.connect(self._show_menu)
    
    def _show_menu(self, pos):
        menu = QMenu()
        index = self.table.indexAt(pos)
        
        if index.isValid():
            row = index.row()
            menu.addAction("📋 Copy Cell", lambda: self._copy_cell(row, index.column()))
            menu.addAction("📋 Copy Row", lambda: self._copy_row(row))
            menu.addSeparator()
            menu.addAction("🔍 Filter to This Value", lambda: self._filter_to_value(index))
            menu.addAction("❌ Exclude This Value", lambda: self._exclude_value(index))
            menu.addSeparator()
            menu.addAction("📂 Open in KB", lambda: self._open_in_kb(row))
            menu.addAction("📊 Show Detail", lambda: self._show_detail(row))
        
        menu.addSeparator()
        menu.addAction("📤 Export Visible", self._export_visible)
        menu.addAction("☑️ Select All", self.table.selectAll)
        menu.addAction("☐ Deselect All", self.table.clearSelection)
        
        menu.exec(self.table.viewport().mapToGlobal(pos))
```
