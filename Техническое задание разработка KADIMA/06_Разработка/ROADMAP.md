# 6.2. Delivery Plan / Roadmap — KADIMA

## Phase 1: MVP (Months 1–3)
- Core Pipeline (M1, M2, M3, M8)
- Corpus Management (import, list, export)
- Basic Validation (gold corpus, expected, review)
- CLI interface (no GUI)

## Phase 2: Full Pipeline (Months 4–6)
- Extended Extraction (M4–M7, M12)
- Validation Framework (full)
- Desktop UI (PyQt)
- API (basic)

## Phase 3: Polish & Launch (Months 7–9)
- Annotation Interface (M15)
- TM Projection (M16)
- Advanced UI (comparison, export)
- Documentation
- Marketing site
- Beta testing

## Milestones
- M1: CLI pipeline works on gold corpus → PASS
- M2: Desktop UI usable for basic workflow
- M3: 3 extraction profiles working and comparable
- M4: Validation framework complete
- M5: Public beta
- M6: v1.0 release
